#### GEN-AI-practice
This project is a collection of Python notebooks that demonstrate various concepts in Python programming, such as data types, control flow, functions, classes, and more. The notebooks are organized into different directories, each containing a specific topic.

##### Files
**basics:** This directory contains notebooks that cover basic Python concepts, such as data types, control flow, functions, loops, and more. The notebooks include examples and explanations of each concept.

**files:** This directory contains notebooks that demonstrate how to work with files in Python, such as reading and writing text files, working with CSV files, and more.

**calculator:** This directory contains a simple calculator application built using Python. The application allows the user to perform basic arithmetic operations, such as addition, subtraction, multiplication, and division.

**whatsapp:** This directory contains a notebook that demonstrates how to use the Python library pywhatkit to send automated WhatsApp messages. The notebook includes examples of sending messages, sending images, and more.

**inheritance:** This directory contains a notebook that demonstrates how to use inheritance in Python. The notebook includes examples of creating classes that inherit from other classes, overriding methods, and more.

**lambda:** This directory contains a notebook that demonstrates how to use lambda functions in Python. The notebook includes examples of using lambda functions with map(), filter(), and reduce().

**conditions:** This directory contains a notebook that demonstrates how to use conditional statements in Python. The notebook includes examples of if statements, if-else statements, and nested if statements.

**loops:** This directory contains a notebook that demonstrates how to use loops in Python. The notebook includes examples of for loops, while loops, and nested loops.

**encapsulation:** This directory contains a notebook that demonstrates how to use encapsulation in Python. The notebook includes examples of creating classes with private and public attributes and methods.

**polymorphism:** This directory contains a notebook that demonstrates how to use polymorphism in Python. The notebook includes examples of creating classes with different behaviors but the same interface.

**abstraction:** This directory contains a notebook that demonstrates how to use abstraction in Python. The notebook includes examples of creating classes with abstract methods and properties.

**magic methods:** This directory contains a notebook that demonstrates how to use magic methods in Python. The notebook includes examples of using magic methods for object representation, comparison, and more.
